# Ghostly Candy Quest

**Ghostly Candy Quest** is a spooky-themed adventure game developed using Unity. Players navigate through a haunted world to collect candy and overcome various challenges.

## Features

- Explore a haunted world filled with spooky environments and obstacles.
- Collect candies and other items to progress through the game.
- Simple and intuitive controls for movement and interaction.
- Engaging gameplay with unique challenges and puzzles.

## Technologies

- **C#** (for scripting within Unity)
- **Unity Engine** (for game development and rendering)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/NeRrO106/Ghostly-Candy-Quest.git
   ```

2. Open Unity Hub and add the project folder:
   - Click on "Add" in Unity Hub.
   - Navigate to the `Ghostly-Candy-Quest` directory and add it.

3. Open the project in Unity.

4. Press the "Play" button in Unity to run the game.

## Usage

1. Launch the Unity project from Unity Hub or directly from the Unity editor.
2. Use the in-game controls to navigate through the haunted world.
3. Collect candies and overcome obstacles to complete the quest.
